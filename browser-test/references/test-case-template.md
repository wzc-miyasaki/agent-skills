## 示例：用户登录测试

### 前置条件
- 打开页面：http://localhost:8080/login

### 操作步骤
1. 在【用户名】输入框填写 `admin`
2. 在【密码】输入框填写 `123456`
3. 点击【登录】按钮

### 预期结果
- **页面验证**：页面显示文字"欢迎回来"
- **URL 验证**：页面跳转到 `/dashboard`
- **接口验证**：GET /api/user/info 返回 `{"username": "admin"}`

## 示例：创建订单测试

### 前置条件
- 打开页面：http://localhost:8080/login
- 在【用户名】输入框填写 `admin`
- 在【密码】输入框填写 `123456`
- 点击【登录】按钮
- 等待页面跳转到 `/dashboard`

### 操作步骤
1. 点击导航栏中的【订单管理】
2. 点击【新建订单】按钮
3. 在【商品名称】输入框填写 `测试商品A`
4. 在【数量】输入框填写 `10`
5. 在下拉框 `#supplier-select` 选择 `供应商B`
6. 点击【提交订单】按钮

### 预期结果
- **页面验证**：页面出现提示"订单创建成功"
- **页面验证**：订单列表中出现"测试商品A"
- **接口验证**：POST /api/orders 返回状态码 200，响应体包含 `"status": "created"`
